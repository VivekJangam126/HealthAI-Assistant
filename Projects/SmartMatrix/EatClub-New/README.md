# Eatclub – MERN Stack (Frontend + Backend)

This is a MERN stack rewrite of the existing Eatclub website.

## Tech Stack

- **Frontend (client)**: React, Context API, React Router
- **Backend (server)**: Node.js, Express, MongoDB
- **Other**: Axios (or fetch) for API calls, custom hooks, services, validations

---
