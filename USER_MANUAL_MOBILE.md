## Installation

### Download & Install

**Option 1: From APK File**
1. Download the APK file from the official source
2. Open the APK file on your Android device
3. If prompted, enable "Install from Unknown Sources"
4. Tap "Install"
5. Wait for installation to complete
6. Tap "Open" to launch the app

**Option 2: From Google Play Store** (Coming Soon)
1. Open Google Play Store
2. Search for "AyuMitra"
3. Tap "Install"
4. Open the app

### System Requirements

- **Android Version**: 8.0 (Oreo) or higher
- **Storage**: Minimum 50MB free space
- **RAM**: Minimum 2GB recommended
- **Internet**: WiFi or mobile data connection
- **Permissions**: Camera, Storage, Internet

## Troubleshooting

### App Won't Open

**Solutions**:
1. Restart your phone
2. Clear app cache: Settings â†’ Apps â†’ AyuMitra â†’ Storage â†’ Clear Cache
3. Reinstall the app
4. Check for Android updates

### Camera Not Working

**Solutions**:
1. Grant camera permission: Settings â†’ Apps â†’ AyuMitra â†’ Permissions
2. Close other camera apps
3. Restart the app
4. Restart your phone
5. Check if camera works in other apps

### Images Not Uploading

**Solutions**:
1. Check internet connection
2. Verify storage permission
3. Ensure file size < 10MB
4. Try different image format
5. Clear app cache

### Voice Input Not Working

**Solutions**:
1. Grant microphone permission
2. Check microphone in other apps
3. Ensure no Bluetooth headset connected
4. Restart the app
5. Check Android voice input settings

### App Crashes

**Solutions**:
1. Update to latest version
2. Clear app cache and data
3. Reinstall the app
4. Check available storage (need 100MB+)
5. Report crash to developers

### Slow Performance

**Solutions**:
1. Close background apps
2. Clear app cache
3. Free up phone storage
4. Restart your phone
5. Check internet speed

### Sync Issues

**Solutions**:
1. Check internet connection
2. Logout and login again
3. Clear app data (will delete local data)
4. Verify account credentials

## FAQ

**Q: Is the mobile app free?**  
A: Yes, completely free. You need your own Gemini API key.

**Q: Does it work offline?**  
A: Partially. You can view saved data offline, but AI features need internet.

**Q: How much storage does it need?**  
A: About 50MB for the app, plus space for cached data.

**Q: Can I use it on tablet?**  
A: Yes, works on Android tablets too.

**Q: Is iOS version available?**  
A: Not yet. Coming soon!

**Q: How do I update the app?**  
A: Download and install the latest APK, or update from Play Store (when available).

**Q: Can I use mobile data?**  
A: Yes, but WiFi is recommended to save data.

**Q: How do I backup my data?**  
A: Data is synced to cloud automatically when logged in.

**Q: Can I use multiple devices?**  
A: Yes, login with same account on multiple devices.

**Q: How do I delete my account?**  
A: Contact support@ayumitra.com

**Q: Is my data secure on mobile?**  
A: Yes, same security as web version (HTTPS, encryption).

**Q: Can I share conversations?**  
A: Yes, export as PDF and share via any app.
