## Getting Started

### Accessing AyuMitra

1. Open your web browser (Chrome, Firefox, Safari, Edge)
2. Navigate to: **https://ayumitra.vercel.app**
3. The homepage will load with the main navigation

### System Requirements

- **Browser**: Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- **Internet**: Stable internet connection
- **Screen**: Works on all screen sizes (320px and above)

## Account Management

### Creating an Account

1. Click **"Sign Up"** button in the top right corner

## Settings & Configuration

### Accessing Settings

1. Click your profile icon in the top right
2. Select **"Settings"** from the dropdown
3. Or navigate to **Settings** from the sidebar

### Managing Your Gemini API Key

**Why You Need It**:
- AyuMitra uses Google's Gemini AI
- Each user manages their own API key
- This ensures privacy and control

**How to Get an API Key**:
1. Visit: https://aistudio.google.com/app/apikey
2. Sign in with your Google account
3. Click **"Create API Key"**
4. Copy the key (starts with "AIza...")

**How to Add Your API Key**:
1. Go to Settings
2. Find **"Gemini API Key"** section
3. Paste your API key
4. Click **"Save Changes"**
5. Success! You can now use all AI features

**API Key Status**:
- ‚úÖ **Valid**: Green indicator, all features work
- ‚ö†Ô∏è **Expired/Invalid**: Red indicator, update needed
- ‚ùå **Missing**: Yellow indicator, add key to use features

**Quota Exceeded**:
- If you see "Quota exceeded" error
- Your API key has reached its daily limit
- Get a new API key or wait 24 hours
- Free tier: 20 requests per day per model

### Profile Settings

**Update Display Name**:
1. Go to Settings
2. Edit **"Display Name"** field
3. Click **"Save Changes"**

**Change Password**:
1. Go to Settings
2. Enter current password
3. Enter new password

### Preferences

**Dark Mode**:
- Toggle dark mode from the top navigation

**Language**:

## Troubleshooting

### Common Issues

**"Please configure your API key"**
- **Solution**: Go to Settings and add your Gemini API key
- Get one from: https://aistudio.google.com/app/apikey

**"API key quota exceeded"**
- **Solution**: Your daily limit is reached
- Wait 24 hours or get a new API key
- Free tier: 20 requests/day

**"Network request failed"**
- **Solution**: Check your internet connection
- Refresh the page
- Try again in a few moments

**"Invalid API key"**
- **Solution**: Your API key is incorrect or expired
- Get a new API key
- Update in Settings

**Features not working**
- **Solution**: Ensure you're logged in
- Check if API key is configured
- Refresh the page
- Clear browser cache

**Images not uploading**
- **Solution**: Check file size (max 10MB)
- Use supported formats (JPG, PNG, PDF)
- Ensure good internet connection

**Chat not responding**
- **Solution**: Check API key status
- Verify internet connection
- Try canceling and resending

### Browser Issues

**Clear Cache**:
1. Press Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
2. Select "Cached images and files"
3. Click "Clear data"
4. Refresh the page

**Enable JavaScript**:
1. Go to browser settings
2. Find "Site settings" or "Content settings"
3. Enable JavaScript
4. Refresh the page
